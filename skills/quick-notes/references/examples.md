# Quick Notes Examples

## Simple scratch note

```markdown
# Scratch Note

## 2026-03-26
- Buy batteries
- Check Pi storage
- Ask ISP about upload speed
```

## Project note

```markdown
# Home Server

## 2026-03-26
- Removed Docker and containerd
- Confirmed about 22GB disk free
- OpenClaw gateway remains the main memory user
```

## Raw user wording

If the user says:

> nhớ lưu: đổi git user của Sen thành con_sen

Preserve it closely:

```markdown
# Workspace Notes

## 2026-03-26
- Đổi git user của Sen thành con_sen
```
