# Sen Etsy Playbook

Use this as a practical operating layer on top of the Etsy skill.

## Core Model

Evaluate every Etsy listing in this order:
1. Policy risk
2. Search match
3. Click appeal
4. Offer clarity
5. Conversion trust
6. Margin safety
7. Experiment design

Do not optimize tags alone when the image, title opening, or offer architecture is weak.

## Audit Workflow

### 1. Lock scope fast
Collect only what materially changes the advice:
- Product type
- Target buyer
- Main use case
- Physical vs digital format
- Price range
- Shipping region and processing time
- Current goal: traffic, CTR, favorites, carts, conversion, or AOV

### 2. Classify the bottleneck
- Low impressions/views -> query match or low demand visibility
- Low clicks -> weak first image or title opening
- Low favorites/carts -> weak value proposition or confusing format
- Low conversion -> trust, shipping, price, policy, or proof friction

### 3. Read the listing stack as one system
Review together:
- Main image
- Title first 40 characters
- Tag cluster coverage
- Description opening
- Offer architecture
- Proof/revision process
- Shipping promise

### 4. Score the listing
Score 1-10 for:
- Policy safety
- SEO alignment
- Click appeal
- Offer clarity
- Trust/conversion readiness
- Differentiation

### 5. Recommend changes by impact
Default order:
1. Main image
2. Title opening
3. Tag architecture
4. Description opening
5. How-to-order clarity
6. Material/size comparison
7. Social proof or process proof

### 6. Design one-variable tests
Change one major variable per cycle.
Use 7-14 day windows unless traffic is very high.
Track:
- Views
- Visits
- Favorites
- Add to cart
- Conversion rate
- Revenue per visit

## SEO Structure

Treat Etsy matching as phrase-sensitive.
Optimize as if exact match matters heavily.

Use 13 tags across four roles:
- 5 primary tags: core product phrases
- 4 attribute tags: material, style, size, format
- 2 use-case tags: recipient, occasion, purpose
- 2 long-tail tags: buyer-intent phrases

### Tag rules
- Prefer multi-word phrases over single words
- Keep tags buyer-facing and concrete
- Avoid duplicate stems unless the variation materially changes intent
- Avoid trademarks and franchise-specific terms
- Ensure the title opening contains 2-3 of the strongest phrases naturally

## Copy Output Rules

When writing listing copy:
- Deliver paste-ready text
- Lead with product clarity before hype
- Keep the first 1-2 lines readable and buyer-facing
- Use bullets for features, materials, process, and shipping
- Add a simple How to Order section for personalized products
- Keep emotional language, but not at the expense of clarity

## Fan-Style / IP Safety Heuristic

Safer:
- genre or vibe language
- original layout systems
- collectible or fantasy wording
- custom value anchored in the user's photo or story

Riskier:
- franchise names
- trademarked mechanics or labels
- lookalike layout that reads as a replica
- tags built around famous IP demand

Use inspired-by category signals, not direct brand capture.

## Default Output Template

For a listing audit, provide:
1. Quick diagnosis
2. Funnel bottleneck guess
3. Biggest risks
4. Highest-impact fixes
5. Revised title
6. Revised 13 tags
7. Revised opening paragraph
8. One test to run first
